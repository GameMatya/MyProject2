//------------------------------------------------
//	 HLSLの固定内容
//------------------------------------------------
Texture:t10 = スカイボックス
Texture:t11 = プレフィルタリング済スカイボックス(Diffuse)
Texture:t12 = プレフィルタリング済スカイボックス(Specular)
Texture:t13 = スカイボックスの色対応表
Texture:t14 = 静的シャドウマップ
Texture:t15 = 動的シャドウマップ

cBuffer:b5  = シーン定数バッファ
cBuffer:b6  = メッシュ定数バッファ
cBuffer:b6 = GPUパーティクル用定数バッファ ( コンピュートシェーダーのみ )
cBuffer:b7  = サブセット用定数バッファ
cBuffer:b8  = ライト用定数バッファ
cBuffer:b9  = ポストプロセス用定数バッファ
cBuffer:b10 = ぼかし用定数バッファ
cBuffer:b11 = シャドウ用定数バッファ
cBuffer:b12 = デバッグライン用定数バッファ

頂点バッファ 0 = 自由
頂点バッファ 1 = GPUインスタンシング

//------------------------------------------------
//	 描画順序
//------------------------------------------------
G-Buffer作成 ・ ShadowMap作成(動的モデルのみ)
|
Defferd Lighting(平行光源や点光源)
|
PostEffect(トーンマッピングやブルーム)

//------------------------------------------------
//	 リニアワークフロー
//------------------------------------------------
3Dオブジェクトの描画 (リニア)
|
ブルームエフェクト (リニア)
|
トーンマッピング (sRGBに戻す)


//------------------------------------------------
//	 マルチスレッド
//------------------------------------------------
    オブジェクトの処理	　
プレイヤー、カメラ
|			
プレイヤーの武器、弾
|			
敵、敵の弾		


//------------------------------------------------
//	 処理順番
//------------------------------------------------
更新処理 / ステージ(スタティック)オブジェクトとの押し出し処理
|
当たり判定 ( 押し出し処理 )
|
描画処理


//------------------------------------------------
//	 当たり判定
//------------------------------------------------
ステージ ( キャラクターが乗れるオブジェクト )
>> レイキャスト

その他 ( キャラクターが乗れないオブジェクト(木,キャラクターなど) )
>> 球


//------------------------------------------------
//	 アニメーションのイベント
//------------------------------------------------
0 = 攻撃の判定時間
1 = アニメーションのキャンセル受付時間
2 = エフェクト発生時間
